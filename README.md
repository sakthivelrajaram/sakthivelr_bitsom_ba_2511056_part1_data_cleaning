# Part 1: Data Cleaning

## Business Problem
The dataset contains inconsistencies such as missing values, duplicate records, and formatting issues. The objective is to clean the dataset to make it suitable for further analysis.

## Dataset Used
- raw_orders.csv (provided dataset)

## Tools Used
- Python
- Pandas

## Steps Performed
1. Loaded the dataset using Pandas
2. Checked for missing values
3. Removed duplicate records
4. Handled missing values using forward fill
5. Standardized column names
6. Performed basic data analysis

## Key Outputs
- Cleaned dataset: cleaned_orders.csv
- No duplicate records
- Missing values handled
- Structured and consistent dataset

## Business Insights
- Data quality improved significantly after cleaning
- Clean dataset ensures accurate business decisions
- Duplicate and missing data issues were resolved

## Assumptions
- Missing values were handled using forward fill method
- Dataset structure is consistent after cleaning

## Limitations
- Advanced data imputation techniques were not used
- Some assumptions may not reflect real-world scenarios
